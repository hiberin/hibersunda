HiberSunda API

Sundanese Undak Usuk Basa Dictionary API

This API allows you to learn Undak Usuk Basa in Sundanese language. Access a comprehensive dictionary of Sundanese words with different politeness levels.

RESTful API
JSON Response
CORS Enabled

Endpoints

GET Get All Words

GET https://hibersunda.vercel.app/undakusukbasa

Retrieve the complete dictionary of Sundanese words with all politeness levels.

Try it:

GET Search Words

GET https://hibersunda.vercel.app/undakusukbasa/{substring}

Search for words containing a specific substring in any field (Sorangan, Batur, Loma, Indonesian, or English).

Example:
GET /undakusukbasa/makan

Response Format

All responses are returned in JSON format with the following structure:

{ "words": [ { "sorangan": "string", "batur": "string", "loma": "string", "bindo": "string", "english": "string" } ] }

About Undak Usuk Basa

Undak Usuk Basa is a system of speech levels in Sundanese language that reflects social hierarchy and respect. The dictionary includes:

  • Sorangan: Language for oneself (informal)
  • Batur: Polite language for peers
  • Loma: Formal/high language
  • Bindo: Indonesian translation
  • English: English translation
